Data
====

How do I upload or download data?
---------------------------------

Use storage URLs in workflow ``inputs`` and ``outputs`` for workflow data movement, or use
the ``osmo data`` CLI for direct storage operations. See :ref:`Working with Data <tutorials_working_with_data>`
and :ref:`osmo data <cli_reference_data>`.
